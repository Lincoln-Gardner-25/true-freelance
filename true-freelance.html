True Freelance

Role: Product Manager & Developer Timeline: January 2026

True Freelance is a web application designed to help freelancers understand their true hourly rate by tracking the actual hours worked on projects alongside the money received.

Problem Statement

Many freelancers underestimate the time they spend on projects, leading to an inaccurate understanding of their actual hourly earnings. This can result in underpricing services and burnout from overwork.

Solution

True Freelance provides a simple, elegant interface for tracking:

  • Project names and completion dates
  • Hours worked on each project
  • Money received for each project
  • Automatically calculated hourly rates
  • Monthly earnings summaries and statistics

Key Features

  • Built-in Timer: Track time as you work with an integrated stopwatch
  • Monthly View: See earnings, hours, and average rate by month
  • Cloud Sync: Data synced across devices with Supabase
  • User Authentication: Secure sign-up and login
  • Responsive Design: Works on desktop and mobile

Technology Stack

  • HTML5, CSS3, Vanilla JavaScript
  • Supabase (Authentication & Database)
  • PostgreSQL with Row Level Security
